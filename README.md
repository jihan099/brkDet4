# The border rank of the $4 \times 4$ determinant tensor is twelve

The purpose of the code in this repository is to implement the arguments in the paper [The border rank of the $4 \times 4$ determinant tensor is twelve](https://doi.org/10.48550/arXiv.2510.11051) by Jong In Han, Jeong-Hoon Ju, and Yeongrak Kim. The code is written in Macaulay2 (version 1.26.06). It uses the Macaulay2 package [Permutations](https://macaulay2.com/doc/Macaulay2/share/doc/Macaulay2/Permutations/html/index.html) by Sean Grate and also refers to the paper [New lower bounds for matrix multiplication and $\det_3$](https://doi.org/10.1017/fmp.2023.14) by A. Conner, A. Harper, J. M. Landsberg and their [implementation](https://github.com/adconner/chlbapolar).
