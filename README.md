# The border rank of the $4 \times 4$ determinant tensor is twelve

The purpose of the codes in this repository is to implement the arguments in the paper 'The border rank of the $4 \times 4$ determinant tensor is twelve'. The files can be run by Macaulay2. Codes were written based on the version 1.26.06. The codes use Macaulay2 packages [Permutations](https://macaulay2.com/doc/Macaulay2/share/doc/Macaulay2/Permutations/html/index.html) by Sean Grate.
